# Problem: Sum of Maximum and Minimum Elements in an Array

## Description

Given an integer array `A` of size `N`, find the sum of its maximum and minimum elements.

## Constraints

*   `1 <= N <= 10^5` (Array size)
*   `-10^9 <= A[i] <= 10^9` (Value of elements)

## Input Format

The first argument `A` is an integer array.

## Output Format

Return an integer representing the sum of the maximum and minimum elements of the array `A`.

## Examples

### Input 1:

A = [-2, 1, -4, 5, 3]

### Output 1:

1

### Explanation 1:

The maximum element is 5 and the minimum element is -4. The sum is $5 + (-4) = 1$.

### Input 2:

A = [1, 3, 4, 1]

### Output 2:

5

### Explanation 2:

The maximum element is 4 and the minimum element is 1. The sum is $4 + 1 = 5$.
