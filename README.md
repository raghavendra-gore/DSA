# DSA problems solved

This repository contains solutions to Data Structures and Algorithms (DSA) problems encountered during the Intermediate and Advanced sections of the Scaler Academy course.

There are two main modules:

## Intermediate Module

Each topic in the intermediate section is structured as follows:

*   **Topic Name**
    *   Assignment
    *   Additional Problems

## Advanced Module

The advanced section is structured similarly, covering various topics:

*   **Topic Name**
    *   Assignment
    *   Additional Problems
